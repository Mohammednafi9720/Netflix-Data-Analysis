# Netflix Data Analysis

## Project Overview
This project analyzes Netflix movies and TV shows using Python data analysis libraries.

## Features
- Movies vs TV Shows analysis
- Country-wise content analysis
- Ratings distribution visualization
- Data visualization using charts and graphs

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Output
The project generates visual charts and insights from the Netflix dataset.
